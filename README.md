# To-Do List App

A simple and stylish **To-Do List** web application built with **HTML, CSS, and JavaScript**.
Keep track of your tasks, mark them as completed, edit, delete, and filter them. Tasks are saved in your **browser’s localStorage** so they persist even after refreshing.

---

## Features

* Add new tasks
* Edit existing tasks
* Delete tasks
* Mark tasks as completed ✔️ (with checkmark)
* Filter tasks by **All / Completed / Pending**
* Completed tasks automatically move to the bottom
* Tasks are stored in **localStorage**
* Smooth animations and responsive design

---

## Technologies Used

* **HTML** – Structure of the app
* **CSS** – Styling, animations, and completed task checkmarks
* **JavaScript** – Functionality: add, edit, delete, complete, filter tasks
* **LocalStorage** – Save tasks in browser
